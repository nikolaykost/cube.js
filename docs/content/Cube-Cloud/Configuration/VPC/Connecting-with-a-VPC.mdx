<InfoBox>

To enable VPC support for your account, please [contact Sales][cube-contact].

</InfoBox>

Cube Cloud supports connecting to databases residing in publicly-inaccessible
VPCs. Additional VPCs can also be connected to a Cube Cloud account (at extra
cost); please [contact Sales for more information][cube-contact].
